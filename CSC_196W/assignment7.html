Getting the a7 API to work (Windows)

  1. Install nodejs onto your computer.
  2. In a terminal, navigate to this repository.
  3. Install the following three packages using the command npm install -g packagename:
    • http-server
    • nodemon
    • eslint
    • stylelint
    • @linthtml/linthtml
    • express
  4. In the terminal, run the nodemon command to start the API server.
  5. You can test the API by following the links below. Try using different parameters!
